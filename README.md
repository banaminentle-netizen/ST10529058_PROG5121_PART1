## Project Information
- Login.Java: Handles all validation methods, registration logic and login logic
- Main.Java: Provides console-based interaction with the user, asking for input and displaying feedback messages
- LoginTest.Java: Contains JUnit tests to verify the correctness of validation methods and Login Functionality.

  ---

  ## Features
  ### Username Validation
A username is valid only if:
- It contains an underscore
- It is no more than 5 characters long

  ### Password Validation
A password is valid only if the following:
- It is at least 8 characters long
- It contains at least 1 capital letter
- It contains at least 1 number
- It contains at least 1 special character

---

  ### Cellphone Number Validation
  A Cellphone Number is valid only if:
  - It starts with "+27"
  - There are no more than 12 numbers
